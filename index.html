Propositions and Basic Terms

What is a Proposition?

A proposition is a statement that is either true or false, but not both.

Logic is the science or study of how to evaluate arguments and reasoning. It provides the foundation for mathematical reasoning and computer operations.

Examples:

✓ Proposition: "2 + 2 = 4" (True)

✓ Proposition: "Water boils at 100°C" (True)

✓ Proposition: "The moon is made of cheese" (False)

✗ Not a Proposition: "What time is it?" (Question)

✗ Not a Proposition: "Have a nice day!" (Command)

Key Concepts

Atomic/Simple Propositions: Single statements like "p" or "q"

Compound Propositions: Connected by logical operators (AND, OR, NOT)

Truth Value: Either True (T) or False (F)

Chapter 4

Logical Equivalence & Classification

Tautology

A tautology is a logic statement that is always true, no matter what values its pieces have. The result column in its truth table is always T (true).

Classic Formula: p ∧ ~p (p OR NOT p)

where p: "It is raining"

p~pp ∨ ~p
TFT
FTT

Notice: No matter if p is true or false, p ∧ ~p is always true!

Real-world Example: "It is raining OR it is not raining." This statement is logically always true.

Contradiction

A contradiction is a statement that is always false: it's impossible for it to ever come out true, no matter what.

Classic Formula: p ∧ ~p (p AND NOT p)

where p: "The number is even"

p~pp ∧ ~p
TFF
FTF

Notice: No matter the input, p ∧ ~p is always false.

Real-world Example: "This number is both even AND odd." This can never be true.

Contingency

A contingency is a statement whose truth depends on its components; sometimes it's true, sometimes false. Most real-world statements are contingencies.

Classic Formula: p ∧ ~p (p AND q)

where p: "Student passes the exam"
where q: "Student attends all classes"

pqp ∧ q
TTT
TFF
FTF
FFF

Depending on p and q, p ∧ ~p can be true OR false!

Real-world Example: "A student passes the exam AND attends all classes." This will be true only if a student does both; otherwise, it's false.

Logical Equivalence

Two propositions are logically equivalent if they have identical truth tables—same truth value for all possible combinations.

Example: p → q is equivalent to ~q → ~p
pqp → q~q → ~p
TTTT
TFFF
FTTT
FFTT

✓ These are equivalent (same truth values in all rows)!

Important Logical Equivalences:

  • ~(~p) ≡ p (double negation)
  • p ∧ q ≡ q ∧ p (commutative)
  • p ∨ q ≡ q ∨ p (commutative)
  • (p ∧ q) ∧ r ≡ p ∧ (q ∧ r) (associative)
  • ~(p ∧ q) ≡ ~p ∨ ~q (De Morgan's law)
  • ~(p ∨ q) ≡ ~p ∧ ~q (De Morgan's law)
